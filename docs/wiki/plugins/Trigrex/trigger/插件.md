# 插件

仅列出与当前功能相关参数，更多详见完整配置

## MythicMobs

```yaml
# 击杀怪物
type: kill-mob
# 怪物条件
mob:
  - "恶魔士兵"
# 掉落物
drop:
  - "mm 恶魔士兵碎片 5"
  - "mm 恶魔士兵核心 1~3 0.1"
# 自动拾取
auto-pickup: true
```

## 龙核按键

```yaml
# 按下按键、释放按键
type: dragon-keyboard,dragon-keyboard-end
```

## 萌芽按键

# 按下按键、释放按键
