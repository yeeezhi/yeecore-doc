# 灵魂兑换 - SpaceRingExchange

#### 适用版本

- 1.9-1.12

#### 依赖插件

- SpaceRingPlus
- SimpleLib (售后群下载)

#### 插件购买：

捆绑包1 38元：

- 限时倍率 - SpaceRingItem
- 灵魂兑换 - SpaceRingExchange

捆绑包2 99元：

- 灵魂商店 - SpaceRingShop
- 限时倍率 - SpaceRingItem
- 灵魂兑换 - SpaceRingExchange

#### 🔧插件功能：

- 为 SpaceRingPlus (灵魂空间) 扩展兑换功能
- 允许玩家消耗灵魂空间内的物品并将其兑换到背包中。
