# 椰の战令 - YeeBattlePass - 88R

#### ℹ️适用版本

- 1.9-1.20

#### 🔧插件介绍：

- 支持 JSON / MySQL 存储📁🔍
- 日常任务系统 YeeTask 已适配该插件📜
- 支持 多战令方案🎯，多战令周期📅，多奖励档位🏆
- 战令、奖励与排版方案非耦合设计，实现快速配置插件🚀
- 战令奖励支持YeeCore表达式，极大丰富了插件的扩展性💡
- 奖励档位不只是是基础与进阶，支持按需自定义无限奖励档位🏆
