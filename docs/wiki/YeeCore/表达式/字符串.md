## 字符串处理
用于处理字符串中的表达式和数值
* 语法：`${表达式}`

在动作和条件表达式中使用
```yaml
action:
  - editItem('name','§3炼狱传说+${matchNumber('普通强化')}')
```


在Lore中使用
```yaml
  "这是一个物品":
    material: GOLD_BLOCK
    name: "§e成长进阶奖励"
    lore:
      - "§f"
      - "§7价格:&d3800点券"
      - "§f"
      # 解锁状态
      - "§c${%ybp_grade_{battlepass_id}_pro% ? '§a已解锁' : '§c点击解锁'}"
```
